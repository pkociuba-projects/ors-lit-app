# ORS Lit App

Prosty projekt webowy korzystający z `lit` i `leaflet` (OpenRouteService demo).

- **Instalacja**

  Zainstaluj zależności:

  ```powershell
  npm install
  ```

- **Uruchamianie w trybie deweloperskim**

  ```powershell
  npm run dev
  ```

- **Budowanie produkcyjne**

  ```powershell
  npm run build
  ```

- **Podgląd po buildzie**

  ```powershell
  npm run preview
  ```

- **Struktura projektu (ważne pliki)**

  - `index.html` — punkt wejścia aplikacji
  - `src/` — kod źródłowy TypeScript i komponenty
  - `package.json` — skrypty i zależności

Jeśli chcesz, mogę dodać więcej szczegółów (opis komponentów, jak konfigurować klucz ORS itp.).
